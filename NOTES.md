# Lesson 1

## What is Jekyll?

Jekyll is a static site generator.

It converts Markdown files into HTML pages.

---

## What is Markdown?

Markdown is a lightweight language used to write website content.

Examples:

# Heading

## Sub Heading

- List item

**Bold**

---

## What is YAML?

YAML stores page configuration.

Example:

title: About

permalink: /

image: prof_pic.jpg

---

## What is Git?

Git is a Version Control System.

Workflow:

Edit
↓

git add
↓

git commit
↓

git push
